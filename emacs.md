# Emacs

- C: Control key

- M: Alt key

## Moving the cursor

| Movement | Keys |
| --- | --- |
| one character to the left | C-l |
| one word to the left | M-l |
| one character to the right | C-r |
| one word to the left | M-r |
| one line up | C-p |
| one line down | C-n |
| one paragraph up | M-{ |
| one paragraph down | M-} |

## Marking

| Optiions | Keys |
| --- | --- |
| begin marking | C-SPACE |
| mark everything | C-x h |
| cut marked text | C-w |
| copy marked text | M-w |
